# dotfiles

## Prerequisite
* Neovim
* Python3

```bash
$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
$ brew install neovim
$ brew install python3
$ pip3 install neovim
```

## Installation
```bash
$ cd ~
$ git clone https://github.com/hiroki-sawano/dotfiles.git
$ sh ~/dotfiles/setup.sh
```
